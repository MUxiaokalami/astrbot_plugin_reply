# AstrBot QQ群自定义关键词回复插件

一个功能强大的QQ群自定义关键词回复插件，支持文字、图片混合回复，群组独立配置，关键词管理，@用户回复，配置热切换。

## 功能特性

- ✅ **多格式回复**：支持纯文字、纯图片、图文混合回复
- ✅ **群组独立配置**：每个群组可以有自己的关键词回复配置
- ✅ **精确匹配**：关键词精确匹配，避免误触发
- ✅ **@用户回复**：回复时可选择是否@触发用户
- ✅ **配置热切换**：管理后台配置实时生效
- ✅ **权限管理**：支持设置管理员QQ号
- ✅ **数量限制**：可设置每个群组最大关键词数量
- ✅ **启用/禁用**：可临时禁用某个关键词回复
- ✅ **格式保持**：完全保留用户输入的原始格式，包括换行和空格

## 安装方法

1. 将插件文件放入 AstrBot 的 `plugins` 目录
2. 重启 AstrBot 或重载插件
3. 在管理后台配置插件参数

## 配置说明

在AstrBot管理后台可以配置以下参数：

| 参数名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| admin_qq | string | "" | 管理员QQ号列表，用逗号分隔 |
| default_enabled | bool | true | 新添加的回复是否默认启用 |
| group_separate | bool | true | 是否启用群组独立配置 |
| max_keywords_per_group | int | 50 | 每个群组最大关键词数量 |
| enable_image_reply | bool | true | 是否启用图片回复功能 |
| allow_network_images | bool | true | 是否允许使用网络图片 |
| reply_with_at | bool | true | 回复时是否@用户 |

## 使用命令

### 添加回复 ：/添加回复 关键词|回复内容
### 添加混合回复 ：/添加回复 关键词|文字内容，[图片]https://example.com/image.jpg
### 添加图片回复 ：/添加回复 关键词|[图片]https://example.com/image.jpg
### /查看回复
### /启用回复 关键词
### /禁用回复 关键词
###/重载配置

## 图片回复说明

插件支持本地图片和网络图片：

- **本地图片**：`[图片]path/to/image.jpg`
- **网络图片**：`[图片]https://example.com/image.jpg`

## 回复格式

- 完全保留用户输入的原始格式
- 支持多行文本，换行符会被正确显示
- 支持图文混排
- 可选择是否@触发用户

## 权限要求

- **添加回复**：需要管理员权限
- **删除回复**：需要管理员权限  
- **启用/禁用回复**：需要管理员权限
- **重载配置**：需要管理员权限
- **查看回复**：所有用户均可使用
