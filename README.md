# AstrBot 自定义回复插件

一个功能丰富的自定义关键词回复插件，支持文字、图片、正则表达式匹配。

## 功能特性

- ✅ 文字回复
- ✅ 图片回复 (JPG/PNG)
- ✅ 正则表达式匹配
- ✅ 精确/模糊匹配模式
- ✅ 启用/禁用规则
- ✅ 管理员权限管理

## 安装方法

1. 将 `plugin_reply.py` 放入你的 AstrBot 插件目录
2. 重启 AstrBot 服务
3. 使用命令进行配置

## 使用说明

### 基础命令

- `/添加回复 关键字|类型|内容` - 添加回复规则
- `/查看回复` - 查看所有规则
- `/删除回复 序号` - 删除指定规则
- `/启用回复 序号` - 启用规则
- `/禁用回复 序号` - 禁用规则

### 回复类型

- **文字**: `关键字|文字|回复内容`
- **图片**: `关键字|图片|图片URL或路径`
- **混合**: `关键字|混合|文字内容|图片URL`
